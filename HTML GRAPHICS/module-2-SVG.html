SVG ( Scalable Vector Graphics ) ( Grafik Vektor yang dapat Diskalakan )

SVG mendefinisikan grafik berbaris vektor dalam XML, yang dapat langsung disematkan di halaman HTML.

Grafik SVG dapat diskalakan, dan tidak kehilangan kualitas jika diperbesat atau diubah ukurannya :

SVG didukung oleh semua peramban utama.

APA ITU SVG?

  • SVG adalah singkatan dari Scalable Vector Graphics
  • SVG digunakan untuk mendefinisikan grafik berbasis vektor untuk web
  • SVG mendefinisikan grafik dalam format XML
  • Setiap elemen dan atribut dalam file SVG dapat dianimasikan
  • SVG adalah rekomendasi W3C
  • SVG terintegrasi dengan standart lain, seperti CSS, DOM, XSL dan JavaScript

Elemen < svg >

Elemen HTML < svg > adalah wadah untuk grafik SVG.
SVG memiliki beberapa metode untuk menggambar jalur, persegi panjang, lingkaran, poligon, teks, dan banyak lagi.

Perbedan Antara SVG dan Canvas

Svg adalah bahasa untuk menggambarkan grafik 2D dam XML, Sementara Canvas menggambarkan grafik 2D secara cepat ( dengan JavaScript ) .
SVG berbasis XML, yang berarti bahwa setiap elemen tersedia dalam SVG DOM. Anda dapat melampirkan pengendali peristiwa JavaScript ke grafik SVG.
Dalam SVG, Setiap bentuk yang digambarkan akan diingat sebagai sebuah objek. jika atribut objek SVG diubah, browser dapat secara otomatis merender ulang bentuk tersebut.
kanvas ditampilkan piksel demi piksel. Di kanvas, setelah grafik digambar, grafik tersebut akan dilupakan oleh browser. jika posisinya harus diubah, seluruh pemandangan perlu digambar ulang, termasuk objek apa pun yang mungkin telah tertutup oleh grafik.

Perbandingan SVG dan Canvas

Tabel di bawah menunjukkan beberapa perbedaan penting antara Canvas dan SVG :

    Svg
  • Resolusi independen
  • Dukungan untuk event handler
  • Kemampuan rendering teks yang baik
  • Rendering lambat jika rumit
  • Tidak cocok untuk aplikasi game
    Canvas
  • Bergantung pada resolusi
  • Tidak ada dukungan untuk event handler
  • Kemampuan rendering teks yang buruk
  • Anda dapat menyimpan gambar yang dihasilkan sebagai .png atau .jpg
  • Sangat cocok untuk game yang membutuhkan grafis intensif
