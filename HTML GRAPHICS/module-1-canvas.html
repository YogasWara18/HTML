HTML Canvas Graphics

Elemen HTML < canvas > digunakan untuk menggambar grafik pada halaman web.

Grafik tersebut memperlihatkan empat elemen : Persegi panjang, Persegi panjang gradien, Persegi panjang mumtiwarna, dan teks multiwarna.

Apa itu HTML Canvas?

Elemen HTML < canvas > digunakan untuk menggambar grafik, dengan cepat, melalui JavaScript.

Elemen ini < canvas > hanya merupakan wadah untuk grafik. Anda harus menggunakan JavaScript untuk benar-benar menggambar grafik.

Canvas memiliki beberapa metode untuk menggambar jalur, kotak, lingkaran, teks, dan menambahkan gambar.

Canvas didukung oleh semua peramban utama.
