The readonly attribute ( Atribut hanya baca )

Atribut input "readonly" menentukan bahwa bidang input bersifat baca-saja.
kolom input baca-saja tidak dapat diubah (namun, pengguna dapat menavigasi ke sana, menyorotnya, dan menyalin teks dari sana ).
Nilai bidang input hanya-baca akan dikirim saat mengirimkan formulir!

The disabled Attribute ( Attribut yang dinonaktifkan )

Atribut input "disabled" menetukan bahwa bidang input harus dinonaktifkan
kolom input yang dinonaktifkan tidak dapat digunakan dan tidak dapat diklik.
Nilai bidang input yang dinonaktifkan tidak akan dikirim saat mengirimkan formulir!

The size Attribute ( Atribut Ukuran )

Atribut input Size menentukan lebar yang terlihat, dalam karakter, dari bidang input.
Nilai default untuk Size adalah 20.
Catatan : Atribut ini Size berfungsi dengan jenis input berikut: teks, pencarian, tel, url, email, dan kata sandi.

The maxlength attribute ( Atribut maxlength )

Atribut input maxlength menentukan jumlah karakter maksimum yang diizinkan dalam bidang input.
Catatan : Bila a maxlength ditetapkan, kolom input tidak akan menerima lebih dari jumlah karakter yang ditentukan. Namun, atribut ini tidak memberikan umpan balik apa pun. Jadi, jika anda ingin memberi tahu pengguna, Anda harus menulis kode JavaScript.

The min and max Attributes

Input min dan max atribut menentukan nilai minimum dan maksimum untuk bidang input.
Atribut min dan max berfungsi dengan tipe input berikut: angka, rentang, tanggal, tanggal waktu lokal, bulan, waktu, dan minggu.
Kiat : Gunakan taribut max dan min secara bersamaan untuk membuat rentang nilai legal.
