Kiat : Gunakan taribut max dan min secara bersamaan untuk membuat rentang nilai legal.

The step Attribute ( Step Attribute )

Atribut input step menentukan interval nomor legal untuk bidang input.
Contoh : jika langkah="3", angka yang sah bisa berupa -3, 0, 3, 6, dst.
Tip : atribut ini dapat digunakan bersama dengan atribut max dan min untuk membuat rentang nilai legal.
Atribut ini step berfungsi dengan tipe input berikut : angka, rentang, tanggal, datetime-lokal, bulan, waktu, dan mimggu.

The height and width attributes ( Attribut Tinggi dan Lebar )

input height dan width atribut menentukan tinggi dan lebar suatu ( ) elemen.

Kiat :

Selalu tentukan atribut tinggi dan lebar untuk gambar. Jika tinggi dan lebar ditetapkan, ruang yang diperlukan untuk gambar akan dicadangkan saat halaman dimuat. Tanpa atribut ini, browser tidak mengetahui ukuran gambar, dan tidak dapat menyediakan ruang yang sesuai untuknya. Efeknya adalah tata letak halaman akan berubah selama pemuatan ( saat gambar dimuat ).

The Autocomplete Attribute ( Atribut pelengkapan otomatis )

Atribut input autocomplete mentukan apakah formulir atau bidang input harus mengaktifkan atau menonaktifkan pelengkapan otomatis.
pelengkapan otomatis memungkinkan browser untuk memprediksi nilai. saat pengguna mulai mengetik di kolom, browser akan menampilkan opsi untuk mengisi kolom, berdasarkan nilai yang diketik sebelumnya.
Atribut ini autocomplete berfungsi dengan jenis < form > berikut < input > ; teks, pencarian , url, tel, emeil, kata sandi, pemilih tanggal, rantang, dan warna.
