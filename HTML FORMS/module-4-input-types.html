Tipe input Tersembunyi

Mendefinisikan bidang input tersembunyi ( tidak terlihat oleh pengguna ) .

Bidang tersembunyi memungkinkan pengembang web menyertakan data yang tidak dapat dilihat atau diubah oleh pengguna saat foemulir dikirimkan.
Bidang tersembunyi sering kali menyimpan catatan basis data apa yang perlu diperbarui saat formulir dikirimkan.
Catatan :Meskipun nilainya tidak ditampilkan kepada pengguna dalam konten halaman, nilainya terlihat (dan dapat diedit) menggunakan alat pengembang browser atau fungsi "lihat sumber". jangan gunakan input tersembunyi sebagai bentuk keamanan!

Contoh

Pembatasan input

Berikut adalah daftar beberapa batasan input umum :

Attribute Description
cheked Menentukan bahwa bidang input harus dipilih terlebih dahulu saat halaman dimuat (untuk type="checkbox" atau type="radio")
disabled Menentukan bahwa bidang input harus dinonaktifkan
Max Menentukan nilai maksimum untuk bidang input
Maxlength Menentukan jumlah karakter maksimum untuk bidang input
Min Menentukan nilai minium untuk bidang input
Pattern Menentukan ekspresi reguler untuk memeriksa nilai input terhadap
Readonly Menentukan bahwa bidang input hanya dapat dibaca ( tidak dapat diubah )
Required Menentukan bahwa kolom input diperlukan ( harus disii )
Size Menentukan lebar ( dalam karakter ) bidang input
Step Menentukan interval nomor legal untuk bidang input
Value Menentukan nilai default untuk bidang input

Anda akan mempelajari lebih lanjut tentang pembatasan input pada bab berikutnya.

Contoh berikut menampilkan kolom input numerik, tempat anda dapat memasukkan nilai dari 0 hingga 100, dalam langkah 1.0 Nilai default adalah 30 :

Contoh
