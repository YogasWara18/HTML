HTML Input Form* Attributes

Bab ini menguraikan berbagai form* atribut untuk elemen HTML < input >

The Form Attribute ( Atribut Formulir )

Atribut input form menentukan < input > tempat elemen tersebut berada.
Nilai atribut ini harus sama dengan atribut id elemen < form > yang menjadi miliknya.
