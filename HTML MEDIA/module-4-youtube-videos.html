VIDEO HTML YOUTUBE

Cara termudah unutk memutar video dalam HTML adalah dengan menggunakan YouTube.

Kesulitan dengan Format Video?

Mengonversi video ke format lain bisa jadi sulit memakan waktu.
Solusi yang lebih mudah adalah membiarkan YouTube memutar video dihalaman web Anda.

ID Video YouTube

YouTube akan menampilkan id ( seperti tgbNymZ7vqY ), saat Anda menyimpan ( atau memutar ) video.
Anda dapat menggunakan id ini, dan merujuk ke video anda dalam kode HTML.

Memutar Video Youtube dalam HTML

Untuk memutar video anda di halaman web, laukan hal berikut :

  • Unggah video ke YouTube
  • Catat ID videonya
  • Tentukan < iframe > elemen dihalaman web anda
  • Biarkan src atribut menunjuk ke URL video
  • Gunakan atribut width dan height untuk menentukan dimensi pemain
  • Tambahkan parameter lain ke URL

Putar Otomatis + Bisukan YouTube

Anda dapat membiarkan video anda mulai diputar secara otomatis saat pengguna mengunjungi halaman tersebut, dengan menambahkan autoplay=1 URL YouTube.Namun, memulai video secara otomatis akan mengganggu pengunjung Anda!

Catatan: Peramban Cgromium tidak mengizinkan pemutaran otomatis dalam kebanyakan kasus. Namun, pemutaran otomatis tanpa suara selalu diizinkan.

Tambahkan mute=1 setelah autoplay=1 untuk membiarkan video Anda mulai diputar secara otomatis ( tetapi tanpa suara ) .

Daftar Putar YouTube

Daftar video yang akan diputar yang dipisahkan koma ( selain URL asli ).

Putaran YouTube

Tambahkan playlist=videoID dan loop=1 untuk membiarkan video anda berulang selamanya.
loop=0 (default) - video akan diputar satu kali saja.
loop=1 - Video akan berulang ( Selamanya ).

YouTube - Putar terus menerus

Kontrol Youtube

Tambahkan controls=0 untuk TIDAK menampilkan kontrol di pemutar video.
controls=0 - Kontrol pemutar tidak ditampilkan
controls=1 ( default ) - Kontrol pemutar ditampilkan.
