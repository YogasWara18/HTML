Temukan Posisi Pengguna

API Geolokasi digunakan untuk mengakses lokasi pengguna saat ini.
Karena hal ini dapat membahayakan privasi, lokasi tidak tersedia kecuali pengguna menyetujui.

Catatan : API Geolokasi hanya tersedia pada konteks aman seperti HTTPS.

Kiat : API Geolokasi paling akurat untuk perangkat dengan GPS, seperti telepon pintar atau jam tangan pintar.

Menggunakan API Geolokasi HTML

API Geolokasi diakses melalui panggilan ke navigator.geolocation. Ini akan menyebabkan browser meminta izin kepada pengguna untuk mengakses data lokasi mereka. Jika pengguna menerima, browser akan mencari fungsionalitas terbaik yang tersedia pada perangkat untuk mengakses informasi ini ( misalnya GPS ).

Metode ini getCurrentPosition() digunakan untuk mengembalikan lokasi pengguna saat ini.

Contoh yang dijelaskan :

  • Periksa apakah Geolokasi didukung
  • Jika Geolokasi didukung, jalankan getCurrentPosition() metode tersebut. Jika tidak, tampilkan pesan kepada pengguna
  • Fungsi succes() menampilkan lokasi pengguna dalam garis lintang dan garis bujur
  • Fungsi error() memberi peringatan teks jika browser mengambil kesalahan di getCurrentPosition()
