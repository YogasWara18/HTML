HTML - Apa itu Web API?

Web API adalah impian pengembang.

  • Ini dapat memperluas fungsionalitas browser
  • Ini dapat sangat menyederhanakan fungsi yang kompleks
  • Ini dapat memberikan sintaksis yang mudah untuk kode yang kompleks

Apa itu Web API?

API adalah singkatan dari Application Programming Interface. API adalah semacam antarmuka yang mencakup serangkaian fungsi dan subrutin yang memungkinkan programmer mengakses fitur atau data tertentu dari suatu aplikasi, sistem operasi, atau layanan lainnya.

Web API adalah antarmuka pemrograman aplikasi untuk web.

API HTML

Semua browser memiliki seperangkat API web bawaan untuk mendukung operasi yang kompleks, dan untuk membantu mengakses data.
Berikut adalah beberapa API HTML5 utama :

1.Geolocation API- API ini digunakan untuk mengakses lokasi pengguna saat ini ( dengan lintang dan bujur )
2.Drag and Drop API- API ini memungkinkan anda menggunakan fitur drag-and-drop di browser.
3.Web Stroge API- API ini memiliki mekanisme untuk memungkinkan browser menyimpan pasangan kunci/nilai ( dengan cara yang lebih intuitif daripada cookie ).
4.Web Workers API- API ini memungkinkan JavaScript berjalan dilatar belakang, tanpa mempengaruhi kinerja halaman. Pengguna dapat terus melakukan apa pun yang mereka inginkan : mengklik, memilih sesuatu, dll. Sementara web worker berjalan di latar belakang.
5.Server-Sent Events API- API ini memungkinkan halaman web untuk secara otomatis mendapatkan pembaruan dari server.
6.Canvas API- API ini memungkinkan anda menggambar grafik, dengan cepat, melalui JavaScript.

Catatan tentang penggunaan API HTML

Saat mengimplementaskan API HTML, Anda harus selalu:

Periksa Kemampuan Peramban - Selalu periksa apakah peramban target mendukung API. Selalu tambahkan skrip atau pesan yang akan dijalankan jika peramban tidak mendukungnya.
Tambahkan Penanganan Kesalahan yang kuat - Tambahkan penanganan kesalahan yang kuat untuk menangani skenario dimana API mungkin tidak berfungsi seperti yang diharapkan, untuk memastikan pengalaman pengguna yang lancar.
Minta Izin Pengguna - Saat menggunakan API yang mengakses data sensitif, seperti API Geolokasi ( yang akan mengakses posisi geografis pengguna ), selalu minta persetujuan pengguna sebelum melanjutkan.

API PIHAK KETIGA

API pihak ketiga tidak terpasang di peramban Anda.
Untuk menggunakan API ini, Anda harus mengunduh kode dari Web.

Contoh :

  • API YouTube - Memungkinkan Anda menampilkan video di situs web.
  • Twitter API - Memungkinkan anda menampilkan tweet di situs web.
  • API Facebook - Memungkinkan anda menampilkan info facebook di situs web.
